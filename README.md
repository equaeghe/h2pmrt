# h2pmrt
Python HTML to poor man's rich text converter for mail messages using bs4, tinycss2, and minify-html

## Installation

In module folder:
```
python -m build -w
```

In `dist` subfolder:
```
pip install --user --break-system-packages h2pmrt-0.0.0-py3-none-any.whl
```

Then it should be installed in ` ~/.local/lib/python3.13/site-packages/`.

## Uninstallation

```
pip uninstall --break-system-packages h2pmrt
```

Then it should be removed from ` ~/.local/lib/python3.13/site-packages/`.
